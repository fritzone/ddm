The startup screen

At the startup screen you have the following choices:

Open an existing solution

This will load a solution that you have created in the past, and you can continue your work on it. The quick buttons allow you to load the last projects you worked on.

Create a New Solution

When working with DDM, all your work (i.e. Data Types, Tables, Diagrams, ...) is collected into a Project, which in itself is contained in a Solution. By clicking the "New Solution" button you will create a new solution and an associated project with the possibilty to import a predefined set of data types for the chosen database.

Create a New Connection

DDM has a builtin connection manager, and it stores the connection so that you dont have to type them in every time you need to connect to a database. With the "New Connection" button you can create a new connection. For a detailed description of the connections please see this.

You can close the startup screen using the "Esc" button.

In case you don't want to see the startup screen again check the "Don't show this again" checkbox.

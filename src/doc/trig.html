Triggers

Triggers are database objects that are associated with database tables and that activate when a particular event occurs for a table. These events can be one of:

- INSERT
- DELETE
- UPDATE

A trigger can be set to activate either BEFORE or AFTER the specified event. On the specific event the SQL statements in the triggers "body" gets executed. You can specify either a one line trigger body or a multiline, in this case enclose in BEGIN/END.

With DDM you can create a trigger by specifying the SQL body of the trigger with a normal SQL editor and by choosing the event and trigger time from a list of predefined values.

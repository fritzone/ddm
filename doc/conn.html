Connections

DDM supports the notion of "Connection" to a database. For you this basically means that you don't have to type in the database details every time you want to deploy something to a machine, but only once, when you create the connection, and from that point the connection is available for future use.

To create a connection click the button in the application main toolbar and fill in the required details:
- Name (the name of the connection)
- Host (the host where this database is located)
- User/Password (the required authentication data - Warning!!! The Password will be stored in plaintext in the connection configuration file)
Now, click Connect.
The combo box of databases will be populated with the databases found at the given location. Or you can create a new database by clicking the "+" button. Check the "Auto Connect" checkbox if you want to start this connection every time you start DDM.

From this point your connection is remembered for future use. You have all your connections listed in the Conenction tree (accessible from the "View" menu) where you can either test the connection or delete or modify them.

DDM offers the possibilty to browse the database objects existing in your connections. For more about this please read this document.

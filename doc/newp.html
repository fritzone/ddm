New Solution

Create a new solution and a default project.

Solution Name: Give a descriptive name for the solution.

Create a default project: A new solution for v0.1a always needs to have a default project, so make sure this checkbox is checked.

Project Name: Give a descriptive name for the project. Or you can leave this to be “default”.

Project type combo-box:
Datamodel Project: create a project in which you will create the data types and tables required to deploy a new project.

Database combo-box:
MySQL: the only database supported by v0.1a of DDM

Inherit default datatypes: by checking this you will create a few datatypes that will help you create a basic table layout, you are allowed to create more datatypes to suit your needs

Allow Object Oriented Modelling: by checking this you will create a project which supports table templates which allow you to create one or more table instances using the same layout.

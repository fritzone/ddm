New Solution

Create a new solution and a default project.

Solution Name: Give a descriptive name for the solution.

Create a default project: A new solution for the current version always needs to have a default project, so make sure this checkbox is checked, otherwise you won't be able to proceed.

Project Name: Give a descriptive name for the project. Or you can leave this to be “default”.

Project type combo-box:
Datamodel Project: create a project in which you will create the data types and tables required to deploy a new project.
Reverse engineer a database: create a project in which you will populate the tables and other database entities from an already existing database.

Database combo-box:
the only database supported by this version of DDM is MySql.

Inherit default datatypes:by checking this you will create a few datatypes that will help you create a basic table layout, you are allowed to create more datatypes to suit your needs. This will speed up your database design, however it is highly reccomended that you create your own data types for your project.

AllowObject Oriented Modelling: by checking this you will create a project which supports table templates which allow you to create one or more table instances using the same layout.

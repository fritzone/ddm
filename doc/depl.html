Deploying the solution

Deploying the solution in terms of DDM means to take the generated SQL scripts of all the components of the solution, connect to a specific database and run the SQL script. This will create the tables, views and foreign keys. Deployment can be done by:

- Clicking the "Deploy solution" button in the main toolbar. This will open a dialog box with all the connection that are defined and you can select one or more connections. DDM will deploy the solutoin to all the selected databases.
- Long click the "Deploy solution" button and in the popup menu select the database you want this solution to be deployed.

When starting the deployment DDM will update the status bar of the application, and when done with the deployment DDM will give a feedback about the status of the deployment.

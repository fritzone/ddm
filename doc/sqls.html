SQLs

This screen contains the SQLs generated for the selected item on the project tree: If you have clicked on the "SQL" item, it contains the SQLs for all the tables of the project (or table instances in case of an OOP project), if you have clicked on an item under the SQL item (such as a table, or a table instance) it contains the SQL for that item.

You have the possibility to save the SQL to a file, or inject the SQL in an existing database.

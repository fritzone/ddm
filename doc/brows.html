Browsing a connection

After you have created your connections you can browse them. To do this, either

1. Right click the connection in the connection tree and select "Connect" then Right click again and select "Browse" (you only can browse a connected connection)
2. Double click the connection in the connection tree, this will automatically connect the connection and starts browsing it.

After these steps now you are able to see the tables, views, stored methods and triggers of the database represented by the selected connection. If you don't have a solution you just can see the data stored in the table. For this, just double click the table (or right click and choose "Browse table data").

If you have a solution you can import a table from the database into your solution, without the need to do a full reverse engineering of the database and you also have the possibilty to add tables from different databases to your solution, and deploy them later.

Other options regarding connections

When you right click a connection you will get a menu where the followings are the options you can use:

Run Script

By selecting this option an SQL editor window bound to the database specified in this connection will open. In this window you can type in SQL queries and execute them using Ctrl + Enter. Selecting this option automatically connects the database, starts browsing it and shows the content of the database.

Connect

Selecting this option verifies whether the connection to the database is possible or not. This option does not start browsing the connection. If the database connection is possible the icon is updated with a green signaller, otherwise it gets updated with a red one.

Browse

Selecting this option will try to connect to the database represented by the connection object. Depending on the results of the operation it changes the icon of the connection. In case of success the database objects are displayed in the tree of the connection item.

Edit

Will open the window to modify the properties of the connection.

Recreate

Choosing this option will execute a "DROP DATABASE" and "CREATE DATABASE" script for the database represented by the given connection.

Delete

Choosing this option will remove the connection from the list of available connections, this does not affect the physical database.

Drop

Choosing this option will execute a "DROP DATABASE" on the given connection.

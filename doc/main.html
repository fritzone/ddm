Welcome to DDM - The Database Deployment Manager

DDM is a unique tool for getting into control in the database creation and initial setup process by helping professionals to have consistent database layouts and delivery between projects, sites and versions.

At the startup screen you have the following choices:

Create a New Solution

When working with DDM, all your work (i.e. Data Types, Tables, Diagrams, ...) is collected into a Project, which in itself is contained in a Solution. By clicking this button you will create a new, empty solution and an associated project with the possibilty to import a predefined set of data types for the chosen database.

Open an existing solution

This will load a solution that you have created in the past, and you can continue your work on it.

Create a New Connection

DDM has a builtin connection manager, and it stores the connection so that you dont have to type them in every time you need to connect to a database. With this button you can create a new connection. For a detailed description of the connections please see this.

You can close the startup screen using the "Esc" button.

Note: From version 0.1g DDM offers the possibilty to browse your connections. For more about this please read this document.

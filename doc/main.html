Welcome to DBM - The Database Manager Application


DBM is a unique tool for getting into control in the database creation and initial setup process by helping professionals to have consistent database layouts and delivery between projects, sites and versions. In case you need further help consult the available help system or please feel free to contact the developers.

At this screen you have two choices:

Create a New Solution

When working with DBM, all your work (i.e. Data Types, Tables, Diagrams, ...) is collected into a Project, which in itself is contained in a Solution. By clicking this button you will create a new, empty solution and an associated project with the possibilty to import a predefined set of data types for the chosen database.

Open an existing solution

This will load a solution that you have created in the past, and you can continue your work on it.

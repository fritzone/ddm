Data type

Properties of the data type

Name: Give a unique name for the data type. A small advice is to follow some consistent notation among your datatypes.

Type: A datatype can be either a number, or a string, a boolean, a date or a.blob. Specify here the type of teh datatype

Underlying SQL: Select the SQL type this datatype will have. This combobox is populated bassed on the type of the datatype

Width: This is the width of the data-type. For text/blob type DTs this.is the length of the text, for numerical types this is the max number of column digits. Also you can specify a decimal counter, seprated by a comma from the width (such as 10,2). This field is not used for the other types.

Values: In case you want to create a “Misc” datatype which can choose its values from a list (such as ENUM, or SET) here you can add the initial values of the datatype.
